# ChemEng Toolkit

A lightweight chemical engineering toolkit for common calculations such as ideal gas behavior, unit conversion, Reynolds number, pressure drop, and heat duty.

## Structure

- calculators/: engineering calculation modules
- utils/: reusable helpers and constants
- data/: small reference datasets
- tests/: unit tests

## Quick start

```bash
python -m pip install -r requirements.txt
python app.py
```
